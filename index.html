My Working Pattern
Audit, Architect, Govern, Build, Enable AI
I do not write articles and move on. I audit what exists, identify structural and content flaws, design the information architecture, establish style governance, migrate content through phased platform evolution, and then enable AI systems on top of the structured corpus. Every project I have led follows this pattern, from AssetPanda (flat HTML to GitBook to Ursa AI) to Pegasystems (MadCap Flare to DITA via Oxygen XML across 60 global contributors).

Enterprise DocAgent System: 7-Agent Architecture with Claude

A complete implementation guide for deploying end-to-end documentation agents in enterprise environments. Built on Anthropic's Agent Skills open standard and Model Context Protocol (MCP), now under the Linux Foundation's Agentic AI Foundation. The system uses a 4-layer architecture: MCP Tools (75+ connectors), Agent Skills (SKILL.md files with domain expertise), an Orchestrator (claude-opus-4-6 coordinating specialist agents), and a Governance layer (audit logs, human gates, cost controls).

7 Specialist Agents
Intake, Release Notes, User Guide, Online Help, KB, API Docs, QA
Each agent runs claude-sonnet-4-6 with tailored SKILL.md files. Orchestrator uses claude-opus-4-6 for task routing and quality gates.
5 Human Approval Gates
After Intake, After Draft, After QA, Before Publish, Post-Publish Spot-Check
No content advances without human approval. Agents augment writers, they do not replace them.
Governance Framework
Data classification, per-agent least-privilege, audit logging, cost controls
Every agent action logged with timestamp, agent ID, input/output. API keys in secrets manager, rotated quarterly.
Org Assessment Playbook
Content maturity, technical readiness, team size, budget tiers
3-phase implementation roadmap: Pilot (0-90 days), Expansion (3-9 months), Enterprise (9-18 months). ROI benchmarks from documented deployments.

Personal Edition: Zero-Cost Implementation with Local LLMs

Adapted from the enterprise guide. Replaces every paid component with free, open-source equivalents while preserving the full 7-agent architecture. Runs entirely on a local machine with 16GB RAM. Adds a complete DITA and Diataxis style training module not covered in the enterprise edition.
